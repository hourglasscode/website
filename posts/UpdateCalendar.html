I created this script in order to have a visual representation of previous scans from Nexpose. In this example, I have a Google calendar with the following scan history.

Before we proceed with the script we need to get the ID of the Calendar that we will be updating.

  • Open your calendar
  • Click on the small icon with the downward facing arrow
  • Click on Calendar Settings
  • Scroll down and copy the Calendar ID into a temporary document. It should look something like this: "abcd123efg456@group.calendar.google.com"
  • Go back to the Google Sheet
  • The URL should looke something like this: ""docs.google.com/spreadsheets/d/xyz10/edit#gid=1841025902"
  • Copy and paste the xyz10 portion respective to your sheet onto the aforementioned temporary document
  • Also notate the name of the sheet containing the log
  • Click on Tools
  • Click on Script editor and paste the code below
You will need to set the calendarID, spreadsheetID, and sheetName variables. Once you execute the script, your calendar will be updated to reflect the scans and the scan names on the sheet will be highlighted. The highlighting is there to ensure that as you add more scan events to the log, those that have previously been added to the calendar are not duplicated.
